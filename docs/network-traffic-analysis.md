# Network Traffic Analysis

## Overview

This project aims to analyze network traffic to detect suspicious activity and potential security concerns via packet analysis techniques.

## Objective

* Analyze network traffic patterns
* Identify anomalies and malicious behavior
* Understand packet-level communication
* Apply network security monitoring techniques

## Key Activities

* Captured and analyzed network traffic using tools such as Wireshark
* Investigated packet data including IP addresses, ports, and protocols
* Identified unusual traffic patterns and potential indicators of compromise
* Applied filtering and analysis techniques to isolate suspicious activity

## Skills Developed

* Network traffic analysis and monitoring
* Packet inspection and protocol understanding
* Threat detection and anomaly identification
* Use of tools like Wireshark
