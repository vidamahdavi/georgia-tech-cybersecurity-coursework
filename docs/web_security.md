# Web & API Security

## Overview

This project focuses to find and analyze vulnerabilities in web applications and APIs, prioritizing the improvement of application security.

## Objective

- Understand common web vulnerabilities  
- Analyze attack vectors targeting web applications  
- Apply secure development and testing principles  
- Identify risks in APIs and web services  

## Key Activities

- Studied common vulnerabilities based on OWASP Top 10  
- Analyzed web application attack patterns (e.g., injection, XSS)  
- Evaluated API security risks and misconfigurations  
- Explored mitigation strategies and secure coding practices  

## Skills Developed

- Web application security analysis  
- Vulnerability assessment  
- Risk identification and mitigation  
- Secure development awareness
